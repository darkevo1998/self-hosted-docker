- Or SimpleScreenRecorder

### Step 3: Record Demo (Follow This Order)

#### Part 1: Language Selection (1-2 minutes)
1. Click your **profile icon** (top-right)
2. Go to **Settings** or **Preferences**
3. Find **Language** option
4. **Select a language** (e.g., Arabic, Spanish, French)
5. **Show entire interface switching** - point out:
   - All menus change
   - All buttons change
   - No English text remains

#### Part 2: Create Workflow (3-4 minutes)
1. Click **"New Flow"** (in selected language)
2. Show **connectors catalog** - browse pieces (all in selected language)
3. **Add Trigger:** Search for "Gmail" or "Email"
   - Select "New Email" trigger
   - Show configuration in selected language
4. **Add Action:** Search for "Google Sheets"
   - Select "Add Row" action
   - Show configuration form in selected language
5. **Save and Publish** workflow
6. Show workflow is active

#### Part 3: Summary (30 seconds)
- Recap: "Everything is in [selected language]"
- Show key features
- End recording
